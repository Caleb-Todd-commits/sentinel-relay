## Band’s role

For seeded live execution, Band carries room identity, participant routing, shared agent messages, approval continuation, and the collaboration trail. Verified replay is labeled when that path is unavailable.

Do not claim the open-ended `/api/custom-incident` route creates a Band room. Its value is coordinated server-side AI/ML API reasoning across the same specialist roles.

## Strongest moment
